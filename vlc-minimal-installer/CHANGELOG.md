# Changelog - Pi Signage VLC Minimal

## [1.0.0] - 2024-01-02

### Première version
- Installation automatique de VLC
- Configuration du démarrage automatique
- Désactivation de l'économiseur d'écran
- Scripts utilitaires :
  - Contrôle VLC (start/stop/restart/status)
  - Import automatique depuis USB
- Scripts optionnels dans extras/ :
  - Configuration Google Drive
  - Interface web VLC
  - Montage de partage réseau SMB/CIFS
- Script de désinstallation propre
- Support Raspberry Pi OS Desktop (32/64 bits)
- Compatible LXDE et Wayland/X11
